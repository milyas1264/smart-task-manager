# 🚀 Smart Task Manager

A full-stack task management application built with the MERN stack, fully Dockerized and cloud deployed.

---

## 🧠 Tech Stack

### Frontend
- React (Vite)
- Axios
- Context API
- CSS

### Backend
- Node.js
- Express.js
- JWT Authentication
- MongoDB

### Database
- MongoDB Atlas

### DevOps
- Docker
- Docker Compose
- Docker Hub
- GitHub
- Render (Backend Deployment)
- Vercel (Frontend Deployment)

---

## ✨ Features

- User Authentication (Register / Login)
- JWT Protected Routes
- Create Tasks
- Update Task Status
- Delete Tasks
- Priority & Status Control
- REST API Architecture
- Fully Containerized Setup

---

## 🏗 Architecture

User → Frontend (Vercel) → Backend API (Render) → MongoDB Atlas
